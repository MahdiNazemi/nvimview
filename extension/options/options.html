NvimView

Configure when files open in Neovim and how the viewer starts.

General

File Types

JSON array. Each item needs id, label, nvimFiletype, and at least one matcher: extensions, mimeTypes, filenames, or shebangs.

URL Rules

One rule per line. Use *.example.com for host wildcards and url:fragment for URL substring rules.

Neovim

Runs after Neovim opens the file. The extension does not manage panes created by this command; Neovim mappings and resize handlers own that behavior.

Viewer

Project Root

NvimView starts Neovim in the selected root so startup commands and local config can open their own panes. Markers use one line each: highest .git, highest AGENTS.md, highest CLAUDE.md, or nearest package.json.
