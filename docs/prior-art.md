# Prior Art

These projects and documents informed the design. They are references, not vendored
source.

- Firefox WebExtensions `webRequest` and resource type filtering.
- Firefox native messaging for local process integration.
- Firefox JSON viewer and PDF viewer behavior.
- Firefox internally viewable types for XML, SVG, WebP, AVIF, and JXL.
- Firenvim as prior art for browser-to-Neovim native integration.
- VSCode-Neovim as prior art for keyboard ownership and passthrough boundaries.
- GitHub Linguist as prior art for language and extension metadata.

Do not copy or vendor prior-project source without a license review and an
implementation reason.
